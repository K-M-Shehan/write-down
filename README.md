# Azure lab

> Made for a lab in a 3rd year module called "Case Study Project".

I've decided to use a mono-repo for this project so both the frontend (React) and backend will be in the same repository. The reason for using a mono-repo is because the app itself will be extremely basic.

The content itself was made using generative AI, therefore the code may turn out to be suboptimal. The app made for this lab is a notes app, which would require a backend to store the notes maybe.

Note that this lab is testing for understanding of the deployment strategies and not the ability to write a React app or writing an optimal backend with dotnet and hence using generative AI for this tedious task is admissible.
